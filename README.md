# THE SPHERE OF K-MEDI 프로젝트

대한민국 한의학을 세계와 연결하는 글로벌 랜드마크 제안서 (powered by DHU · 대구한의대학교)

인터랙티브 HTML 제안서입니다. PC·모바일 모두 최적화되어 있으며, 스크롤 또는 키보드(↑/↓) 로 슬라이드를 넘길 수 있습니다.

## GitHub Pages 로 공유하기

1. 새 GitHub 저장소를 만들고 이 폴더의 모든 파일을 업로드합니다.
   - `index.html`, `README.md`, `.nojekyll`, 그리고 `assets/` 폴더 전체
2. 저장소 → **Settings → Pages** 로 이동합니다.
3. **Source** 를 `Deploy from a branch`, Branch 를 `main` / `(root)` 로 설정 후 Save.
4. 잠시 후 발급되는 주소(`https://<아이디>.github.io/<저장소명>/`)를 공유하면 됩니다.

> `.nojekyll` 파일은 GitHub Pages 가 `assets/` 폴더를 그대로 서빙하도록 하기 위해 반드시 함께 올려야 합니다.

## 구성

- `index.html` — 제안서 본문 (단일 파일, 폰트는 Google Fonts CDN 사용)
- `assets/` — 시안 이미지 4종, 영상 2종, 영상 포스터 2종

## 비고

- 첨부 영상 2종은 우측 하단 워터마크 심볼을 제거하여 재인코딩했습니다.
- 서체: Noto Sans KR (고딕 계열) — 신뢰감 있는 톤.
